# ## Contributing to Archyx Blog

> **Note:** 这是一个由 AI 驱动（AI-First）的项目。在这里，我们不只是在写代码，我们是在优化“机器的生产力”。

### ### 01. How We Work (The Hybrid Model)
本项目的核心开发者是 AI Agents。作为人类贡献者，你的角色是：
*   **Prompt Orchestrator:** 提交高质量的任务指令给 Trae/Claude。
*   **System Auditor:** 审核 AI 生成的 Laravel/Vue 代码，扮演最后一层安全防线（Final Guardrail）。
*   **Edge Case Hunter:** 寻找 AI 逻辑中的盲点（如：Laravel 路由冲突、Vue 组件状态泄露）。

### ### 02. Contribution Path
1.  **Issue First:** 所有的改动必须始于 Issue。请详细描述需求，AI 会自动扫描并将其转化为任务。
2.  **Instruction over Coding:** 优先尝试通过修改 `CLAUDE.md` 或 `agents.md` 来调整全局行为，而不是手动修复每一行代码。
3.  **PR Protocol:** 
    *   提交 PR 时，请附带你使用的 **Prompt**（如果是通过对话生成的）。
    *   确保所有代码符合 **Extreme Minimalism** 风格。

### ### 03. Code Style & Quality
*   **Laravel 11 Idioms:** 遵循 Laravel 11 的极简哲学，优先使用原生方法。
*   **Logic Density:** 每一行代码都必须有极高的 ROI（投资回报率）。禁止冗余逻辑。
*   **Skeptical Review:** 我们欢迎对 AI 生成的代码持怀疑态度。如果 AI 跑偏了，请直接在评论区指出，并触发重新生成逻辑。

### ### 04. Rewards
贡献者将被记录在 `evolution.md` 的“人类导师”名单中，并优先获得 Awarely 生态的内测权限。
